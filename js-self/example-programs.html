Program 3 Start : Check a given no is prime or not

A prime number is a positive integer that is only divisible by 1 and itself. For example, 2, 3, 5, 7, 11 are the first few prime numbers.

  // take input from the user
const number3 = parseInt(prompt("Enter a positive number: "));
let isPrime3 = true;

// check if number is equal to 1
if (number3 === 1) {
    console.log("1 is neither prime nor composite number.");
}

// check if number is greater than 1
else if (number3 > 1) {

    // looping through 2 to number-1
    for (let i = 2; i < number3; i++) {
        if (number3 % i == 0) {
            isPrime3 = false;
            break;
        }
    }

    if (isPrime3) {
        console.log(`${number3} is a prime number`);
    } else {
        console.log(`${number3} is a not prime number`);
    }
}

// check if number is less than 1
else {
    console.log("The number is not a prime number.");
}

A fibonacci sequence is written as:

Example: 0, 1, 1, 2, 3, 5, 8, 13, 21, ...

    console.log(`The factorial of ${number7} is 1.`);
